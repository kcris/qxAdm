# Auto splitting

## Invoice fields

* title
* sum
* afferent date
* estimated/final
* proforma ref: optional, for final only
* total units

## Auto split column

* amounts (same afferent date for all invoices!! same units!)
  * invoice1, fraction1
  * invoice2, fraction2
* splits
  * commons (divby, percent)
  * counted (divby, [units invoice])
  * divided (divby)
